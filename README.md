# AQI & CO₂ Prediction System

The **AQI & CO₂ Prediction System** is an AI-based environmental monitoring platform that predicts air quality (AQI) and CO₂ levels in real-time. It also generates insightful analysis using AI to help users understand trends and contributing factors.

🚀 **Features**
✨ Real-time AQI and CO₂ prediction based on population, traffic, industrial activities, humidity, and temperature
📊 AI-generated environmental insights, explanations, and recommendations using Gemini
📈 Interactive visualizations
🌐 Supports multiple cities and future date predictions
🖥️ Built with Streamlit for easy interface
